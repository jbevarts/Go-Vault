## Terminology

### Adjacency List
- Used to describe edges of a graph as an array of arrays with length equal to # of vertices.  Each entry starts with a vertex and each proceeding element 
represnts nodes that vertex touches.

```
Example:

3<-1->2->4

adjList = 
[[1,3,2], [2,4], [3], [4]]
```
